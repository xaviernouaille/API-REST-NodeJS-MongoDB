# API-REST-NodeJS-MongoDB

## Routes

```json GET: /users ``` GET all users
```json GET: /users/:username ``` GET a specific user
```json POST: /authent/signup ``` POST signup
```json POST: /authent/signin ``` POST signin 

## Commande / Installation

```node
npm install
npm run start
npm run dev
```
